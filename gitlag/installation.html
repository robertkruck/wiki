Gitlab data is stored in /var/opt/gitlab  Create LVM volume to mount this filesystem.

Create backup on NAS to /var/opt/gitlab-backups/

Firewalld configuration with NAS mount

sudo firewall-cmd --permanent --new-zone=nfs-clients
 sudo firewall-cmd --permanent --zone=nfs-clients --add-source=192.168.1.202
 sudo firewall-cmd --permanent --zone=nfs-clients --add-service=nfs
 sudo firewall-cmd --reload
