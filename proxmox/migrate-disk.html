When creating a new VM, I often times forgot to set the datastore from LOCAL to VMSTORE1.  This can be easily fixed by the following:

  1. Select the VM you want to migrate the disk on.
  2. Under Hardware, select the Hard Disk you want to move.
  3. At the top, click on Disk Action.
  4. Select Move Disk
  5. Select the new location and click OK.
